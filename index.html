Get a password manager

It's the only way to generate and safely store unique and strong passwords for different websites. Using unique, strong passwords helps prevent hackers from using breached/stolen data from one website to access your accounts on other websites (check haveibeenpwned.com to see if you've already been breached). So much safer than using the 1 or 2 passwords from your old MSN account in the noughties...

Options

  • 1Password
  • LastPass
  • Dashlane

Time

10 minutes

Turn on patching (automatic system updates)

Why?

Security patches need to be applied regularly (to update your system) so that when new ways of breaching security are invented you’ve got all the latest security holes plugged, keeping your system at bay.

Enable two-factor authentication (2FA)

Why?

Add a safety net to something you know (e.g. password, mother’s maiden name) with something you have (e.g. your phone - to generate a code).

How to

Go to twofactorauth.org, enter any website name and follow the link to their instructions for enabling two factor authentication. Start with your email.

Time

10 minutes per website
